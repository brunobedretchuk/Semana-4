Exercício 3

Classes

São gabaritos que servirão como modelo para a criação de objetos, que herdarão características (caso sejam definidas) e comportamentos a partir do construtor da classe.

Objetos

São instâncias das classes que podem ser criadas com valores diversos para os atributos herdados.

Atributos

Propriedades de um objeto, suas características.

Métodos

Ações desempenháveis por um objeto, comportamento.
