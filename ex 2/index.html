Exercício 2

Escopo

Funciona como um contexto em que se encontra o código. Dessa forma, é através do escopo que se poderá determinar a acessibilidade de variáveis dentro do javascript, a depender do 'bloco' que se encontram, global ou local.

Variáveis globais são declaradas fora de um bloco

Variáveis locais são declaradas dentro de um bloco

Let, Var e Const

São as três palavras que se utiliza para definir variáveis. Possuem diferenças no que tange o escopo, o 'hoisting' e a reatribuição

Keyword Scope Hoisting Can Be Reassigned Can Be Redeclared
var Function Scope Yes Yes Yes
let Block Scope No Yes No
const Block Scope No No No
